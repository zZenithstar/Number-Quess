# Number-Quess
Number quess game with Python!

 --Purpose--
This game have two play mode:
First mode, we are keeping a number and computer write between of 1-100 and we telling you are correct or go up or go down. If it can't quess the number we win but he did in 7 rights, he win.

Second mode, is the most popular quess game, Computer choose a between of 1-100 and we catching the number.

 --USE LIBRARY'S--
 
+random library

 --PYTHON VERSION:3.13--
